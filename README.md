#nagano_cake

## サイト概要
長野県にある小さな洋菓子店「ながのCAKE」の商品を通販するためのECサイト
### サイトテーマ
どなたでもご利用いただける通販サイト
### テーマを選んだ理由
nstagramのDMやメールでの通販の注文を受けていたが、情報管理が煩雑になったため、管理機能を含んだ通販サイトを開設することにした。
