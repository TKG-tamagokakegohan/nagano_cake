# <nagano_cake>

## サイト概要
長野県にある小さな洋菓子店「ながのCAKE」の商品を通販するためのECサイト
### サイトテーマ
<p>どなたでもご利用いただける通販サイト</p>
### テーマを選んだ理由
<p>InstagramのDMやメールでの通販の注文を受けていたが、情報管理が煩雑になったため、管理機能を含んだ通販サイトを開設することにした。</p>
### ターゲットユーザ
<p>甘いものが好きな人　老若男女</p>
### 主な利用シーン
<p>ネットですぐ購入できます。</p>[
